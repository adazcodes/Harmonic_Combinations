# All combinations
## Just the printouts
All of the combinations of jazz patterns which start on harmony notes from the first 3 notes in the harmonic series: (1, 8, 5). 
With exceptions like patterns that start on 3 must break this rule.

```
1-3-5-8 
downbeats must be 1, 5, or 8
1: (1, 3, 5, 8)
2: (1, 3, 8, 5)
3: (1, 5, 8, 3)
4: (1, 8, 5, 3)
5: (5, 1, 8, 3)
6: (5, 3, 1, 8)
7: (5, 3, 8, 1)
8: (5, 8, 1, 3)
9: (8, 1, 5, 3)
10: (8, 3, 1, 5)
11: (8, 3, 5, 1)
12: (8, 5, 1, 3)
```

```
1-2-3-5
downbeats must be 1 or 5
1: (1, 2, 5, 3)
2: (1, 3, 5, 2)
3: (5, 2, 1, 3)
4: (5, 3, 1, 2)
```


```
3-2-1-7
downbeats must be 3 or 1
1: (3, 2, 1, 7)
2: (3, 7, 1, 2)
3: (1, 2, 3, 7)
4: (1, 7, 3, 2)
```


```
1-3-5-7
downbeats must be 1 or 5
1: (1, 3, 5, 7)
2: (1, 7, 5, 3)
3: (5, 3, 1, 7)
4: (5, 7, 1, 3)
```

```
3-5-7-9
downbeats must be 3 or 5

2: (3, 7, 5, 9)
3: (3, 9, 5, 7)
4: (5, 7, 3, 9)
5: (5, 9, 3, 7)
```
